  1. Monte Carlo integration

The expected value of a function, \(f(x)\), over the interval \([a,b]\) is defined as \(\mathbb{E}[f(x)] = \frac{1}{b-a}\int_a^b f(x)dx\). Use this fact to calculate the approximate value of \(\int_2^5e^{−x}\sin(x)\) using simulation methods. Check your answer using the integrate function in R.
